# 원본 데이터

`서울/`에는 한국관광공사 TourAPI 4.0 원본 JSON이 있습니다. 관광지, 문화시설, 축제공연행사, 여행코스, 레포츠, 숙박, 쇼핑의 7개 유형만 적재하며 음식점 파일은 사용하지 않습니다.

- 원본 JSON은 수정하지 않습니다.
- 파일명은 `서울_<유형>.json` 형식을 유지합니다.
- `emotion` 객체는 16개 감정 수치의 초기값으로 적재합니다.
- 여행코스의 빈 주소는 서버 시작 시 Kakao 역지오코딩으로 산출해 SQLite에만 저장합니다.

데이터 적재 흐름은 [아키텍처 문서](../docs/architecture.md)를 참고하세요.
